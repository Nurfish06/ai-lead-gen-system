# AI Lead Generation & Outreach Automation System

## 🚀 Overview
An end-to-end AI-powered system that automates:
- Lead collection
- Data enrichment
- Personalized outreach
- Follow-ups
- CRM tracking

## 💡 Features
- AI-generated personalized emails
- Automated follow-ups
- Multi-source lead ingestion
- Real-time analytics

## 🧠 Tech Stack
- n8n (workflow automation)
- Node.js (backend API)
- PostgreSQL (database)
- OpenAI (AI personalization)

## 📊 Results
- Automates 90% of outbound sales work
- Handles 100+ leads/day
- Improves reply rates with personalization


## ⚙️ Setup
1. Clone repo
2. Setup environment variables
3. Run Docker
4. Import n8n workflows
